# LSH
文件转载站

## 功能变更记录

### 2026-04-16 删除跑酷游戏功能

**删除原因：**
- 游戏界面卡顿，影响用户体验
- 存在性能问题，可能导致浏览器崩溃
- 与网站主要功能（文件分享）关联性不强

**删除内容：**
1. 移除了跑酷游戏的HTML结构
2. 删除了游戏相关的CSS样式
3. 移除了Phaser.js游戏引擎引用
4. 删除了游戏相关的JavaScript代码
5. 移除了游戏入口链接

**保留功能：**
- 文件分享和下载功能
- AI工具集
- 多语言支持
- 主题切换功能

### 2026-04-16 图像识别分类器.zip 性能问题

**问题描述：**
- 文件大小：82.37 MB
- 文件位置：实验程序 → 工具
- 问题原因：文件大小超过GitHub推荐的50MB限制，导致下载时转圈很久才能进入下载界面

**当前状态：**
- 文件保留在仓库中（暂时）
- 下载链接指向GitHub直链，速度较慢

**待办事项：**
- 寻找合适的外部存储方案（百度网盘或其他）
- 获取外部下载链接后替换为高速下载
- 考虑使用Git LFS优化大文件管理
